# DTI-API
Teste prático DTI Digital

- Tecnologias:
  - Springboot 2;
  - Maven;
  - Banco de dados em memória (H2);
  - Java 8
  - JPA
  - JUNIT 4

- API CRUD para cadastro de clientes com as seguintes funções:
  - Cadastro de clientes
  - Listagem de todos os clientes
  - Busca de clientes pelo código
  - Atualização de cliente
  - Exclusão de cliente
  
  **Débito técnico**
  Devido ao prazo para entrega (24 horas) não foi possível a implementação dos testes de todas as funcionalidades da API.
